# dokata-generator-hubot

My Personal Boilerplate for hubot.


## Installation

```
git clone https://github.com/moqada/dokata-generator-hubot.git
```


### Requirements

```
npm install -g dokata
```


## Usage

```
dokata /path/to/dokata-generator-hubot -o hubot-sample 
cd hubot-sample
npm update --save-dev
```
